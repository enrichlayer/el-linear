# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind. Assume good intent.
- Disagree about technical decisions; don't make it personal.
- Critique code, not contributors.

## Reporting

Email conduct@enrichlayer.com. Reports are confidential. We'll respond within
two business days.
